## ğŸ“ Getting Help

### Common Questions

**Q: How do I start?**
A: Read QUICKSTART.md (5 minutes), then visit http://localhost:3000

**Q: Why isn't voice working?**
A: Use Chrome or Edge. Check browser settings and permissions. Read FEATURES.md for details.

**Q: How do I customize my info?**
A: Read README_FULL.md "Customization" section or FEATURES.md "Modification Guide"

**Q: How do I deploy?**
A: Read DEPLOYMENT.md for step-by-step instructions. Vercel is easiest.

**Q: What if the chatbot doesn't respond?**
A: Check .env.local has correct API key. Check browser console (F12). See QUICKSTART.md troubleshooting.

**Q: Can I change colors?**
A: Yes! Read README_FULL.md "Change Colors" section. Edit Tailwind classes.

**Q: How do I add my projects?**
A: Edit app/components/Projects.tsx. Read FEATURES.md or README_FULL.md for details.

**Q: Is my API key secure?**
A: It's in .env.local (not in code). For production, read DEPLOYMENT.md "Security" section.
